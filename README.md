# 기술
```
- jdk 17: 개발 환경

- spring boot 3.3.x: 개발 프레임 워크

- mysql: 서버 db 

- aws ec2, docker: 서버 배포 

- oauth2 kakao login: 외부 로그인 연동  

- redis: 인증 토큰 저장소

- aws api gateway: api 엔드포인트

- aws lambda: 토큰 인증 필터 

- aws elb: 경로 라우팅 

- aws s3 bucket: 이미지 저장소
```
